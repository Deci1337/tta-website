# TTA-website

Landing page for TikTok Agent - AI-powered TikTok automation app.

## Tech Stack

- **React 18** - UI library
- **Vite** - build tool and dev server
- **Tailwind CSS** - utility-first CSS framework

## Features

- Cosmic animated background with twinkling stars
- Gradient title with smooth animation
- Infinite marquee carousels for features, advantages, prospects
- Responsive design
- Optimized for Vercel deployment

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output in `dist/`

## Deploy to Vercel

1. Push to GitHub
2. Import repo at [vercel.com](https://vercel.com)
3. Deploy
